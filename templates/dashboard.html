How to Use
  1. Join a Telegram group where you want to use music
  2. Start a voice chat in that group
  3. Send .join to join the voice chat
  4. Send .play <YouTube/SoundCloud URL> to play music
  5. Use other commands to control playback
Tip: All commands start with a dot (.) and work only in groups where you have admin rights or voice chat permissions.
Note: Userbot commands are sent as regular messages. Other users can see them.
